# Agent kickoff prompts

One ready-to-paste prompt per subsystem. Hand the repo to an agentic coding tool (e.g. Claude
Code), open it at the repo root, and paste the matching prompt. Each prompt points the agent at
its folder, its brief (`AGENTS.md`), its contract, the mocks, and an objective definition of done
(its contract test goes green).

Run them in parallel — one agent per subsystem. Do Phase 0 (`docs/BUILD-PLAN.md`) first.
